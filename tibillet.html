Incidents résolus

🔴 Incident Production — VPS Komidi
Django plante au chargement de la page d'accueil
Symptôme : le site de billetterie Komidi ne chargeait plus — exception Django à chaque chargement.

Cause : un Client en base de données avait une mauvaise valeur de categorie pour son schema_name — Django ne pouvait pas résoudre le tenant.

Correction : 
docker exec -it lespass_django sh -lc "
python manage.py shell -c \"
from Customers.models import Client
c = Client.objects.get(schema_name='ce13207cf99c412986c8d6fec1943d3d')
c.categorie = 'S'
c.save()
\""
# Site opérationnel immédiatement ✓

CAUSE RACINE : LaBoutik inclus dans docker-compose.yml sans être utilisé

↓

↓

↓
Environnement stable — tous les scripts fonctionnels ✓
